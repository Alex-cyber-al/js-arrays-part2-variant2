# js-arrays-part2-variant2

Практична робота 7.2  
Варіант 2: Алгоритми пошуку

## Опис проєкту

Цей проєкт реалізує різні алгоритми пошуку елементів у масивах з аналізом ефективності.

Реалізовано:

- Linear Search: перевіряє кожен елемент масиву по черзі.
- Binary Search: працює тільки з відсортованим масивом. На кожному кроці він ділить масив навпіл..
- Jump Search: переходить по масиву блоками, а потім виконує лінійний пошук у потрібному блоці.
- Interpolation Search: оцінює приблизну позицію елемента за його значенням.
- Exponential Search: швидко знаходить діапазон, де може бути елемент
- Пошук першого входження
- Пошук останнього входження
- Пошук діапазону значень
- Benchmark швидкості
- Текстовий графік залежності часу від розміру масиву

## Технології

- JavaScript
- Node.js
- Консольне меню
- performance.now() для вимірювання часу

## Як запустити проєкт

1. Встановити Node.js.
2. Відкрити папку проєкту у VS Code.
3. Відкрити термінал.
4. Виконати команду:

```bash
node app.js
```
